# Maxlence Frontend
# Installation
Run the following command to clone the repository
```
git clone https://github.com/Adityakitukale499/maxlence-FE.git
```
Go to directory to install packages
```
npm install
```
# Run the App
Go to directory and start the server
```
cd frontend
npm start
```
